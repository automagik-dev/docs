Usage: omni replay [options] <instanceId>

Replay missed messages for an agent instance

Arguments:
  instanceId               Instance ID or name

Options:
  -s, --since <timestamp>  Replay messages received after this ISO 8601
                           timestamp (default: lastSeenAt, max 24h ago)
  -h, --help               display help for command


  Options:
    -s, --since <timestamp>  Replay messages received after this ISO 8601 timestamp (default: lastSeenAt, max 24h ago)
Examples:
  omni replay wa-main                               # Replay missed messages for wa-main (since lastSeenAt)
  omni replay wa-main --since 2026-02-27T10:00:00Z  # Replay messages received after a specific timestamp
