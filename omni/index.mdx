# Omni

**Omni** is Automagik's unified messaging platform. It connects WhatsApp, Telegram, Discord, Slack, and other channels into a single API with event-driven automations.

<Note>
  Omni documentation is coming soon. Check back for guides on instance management, message routing, automations, and agent integration.
</Note>

## What Omni Does

- **Multi-channel messaging** — send and receive across WhatsApp, Telegram, Discord, and Slack from one API
- **Event-driven automations** — trigger workflows on message events, reactions, and presence changes
- **Agent routing** — connect AI agents to any channel with configurable reply filters
- **Media processing** — transcription, image description, and document extraction built in
